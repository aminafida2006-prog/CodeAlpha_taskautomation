# JPG File Organizer 📁

A task automation script built in Python as part of
the CodeAlpha Python Programming Internship.

##  About
The script scans a folder, finds all .jpg files,
and automatically moves them into a new subfolder
called 'JPG_Images' — keeping your files organized!

##  Key Concepts Used
- os — to navigate folders and check file paths
- shutil — to move files from source to destination
- file handling — to read directory contents
- if-else & loops — to validate and process files

## Features
- User inputs any folder path (or uses current folder)
- Auto-creates 'JPG_Images' destination folder
- Skips duplicate files safely
- Shows a full summary after completion
- Option to run again on another folder

## How to Run
Make sure Python is installed, then run:
    python jpg_file_organizer.py

##  Built With
- Python 3 (no external libraries needed)
